üîó Hyperlinks

1. <a> - Anchor/Link

Syntax:

<a href="https://www.example.com">External Link</a>
<a href="index.html">Internal Link</a>
<a href="mailto:test@example.com">Email Link</a>
<a href="tel:+1234567890">Phone Link</a>
<a href="#section1">Anchor Link</a>

Output:

External Link

Internal Link

Email Link

Phone Link

Anchor Link

Notes:

The foundation of web navigation. href attribute specifies destination. Can link to URLs, files, email, phone, or page sections.

Link with Target Attribute

Syntax:

<a href="https://www.example.com" target="_blank">Open in New Tab</a>
<a href="https://www.example.com" target="_self">Open in Same Tab</a>

Output:

Open in New Tab

Open in Same Tab

Notes:

target="_blank" opens in new tab/window. For security, add rel="noopener noreferrer" with target="_blank".

Link with Download Attribute

Syntax:

<a href="document.pdf" download>Download PDF</a>
<a href="image.jpg" download="my-image.jpg">Download Image</a>

Output:

Download PDF (simulated - no actual file)

Download Image (simulated - no actual file)

Notes:

Forces download instead of navigation. Optional filename can be specified.

üñºÔ∏è Images

2. <img> - Image

Syntax:

<img src="image.jpg" alt="Description of image" width="300" height="200">

Output:

Notes:

Self-closing tag. alt attribute is required for accessibility. width/height can be set via attributes or CSS.

Responsive Image with srcset

Syntax:

<img src="small.jpg" 
     srcset="small.jpg 480w, medium.jpg 800w, large.jpg 1200w"
     sizes="(max-width: 480px) 100vw, (max-width: 800px) 50vw, 25vw"
     alt="Responsive image">

Output:

Notes:

srcset provides multiple image sources for different screen sizes. Browser chooses best option.

üñºÔ∏è Advanced Image Elements

3. <picture> - Picture Container

Syntax:

<picture>
    <source media="(min-width: 800px)" srcset="large.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Responsive picture">
</picture>

Output:

Notes:

More control than img srcset. Can serve different image formats or art direction. img is fallback.

4. <source> - Media Source

Syntax:

<source media="(min-width: 800px)" srcset="large.webp" type="image/webp">

Output:

See usage in <picture> and <audio>/<video> examples.

Notes:

Specifies media resources for <picture>, <audio>, and <video>. Self-closing tag.

5. <figure> - Figure Container

Syntax:

<figure>
    <img src="chart.jpg" alt="Sales chart">
    <figcaption>Q4 Sales Performance Chart</figcaption>
</figure>

Output:

Notes:

Semantic container for images, diagrams, code snippets, etc. with optional caption.

6. <figcaption> - Figure Caption

Syntax:

<figcaption>Caption text for the figure</figcaption>

Output:

See the caption in the figure example above.

Notes:

Must be first or last child of <figure>. Provides accessible description of figure content.

üéµ Audio Elements

7. <audio> - Audio Player

Syntax:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support audio.
</audio>

Output:

Notes:

controls attribute shows play/pause buttons. Multiple sources for browser compatibility. Fallback text for unsupported browsers.

Audio with Attributes

Syntax:

<audio controls autoplay loop muted preload="auto">
    <source src="background.mp3" type="audio/mpeg">
</audio>

Output:

Notes:

autoplay (use carefully), loop (repeats), muted (starts muted), preload (none/metadata/auto).

üé¨ Video Elements

8. <video> - Video Player

Syntax:

<video controls width="400" height="300">
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.webm" type="video/webm">
    Your browser does not support video.
</video>

Output:

Notes:

Similar to audio but for video content. width/height set dimensions. Multiple formats for compatibility.

Video with Poster

Syntax:

<video controls width="400" height="300" poster="thumbnail.jpg">
    <source src="movie.mp4" type="video/mp4">
</video>

Output:

Notes:

poster attribute shows image before video plays. Good for user experience and loading indication.

9. <track> - Text Track

Syntax:

<video controls>
    <source src="movie.mp4" type="video/mp4">
    <track kind="subtitles" src="subtitles.vtt" srclang="en" label="English">
    <track kind="captions" src="captions.vtt" srclang="en" label="English Captions">
</video>

Output:

Text tracks provide subtitles, captions, descriptions, etc. for video/audio content.

Notes:

Self-closing tag. kind: subtitles, captions, descriptions, chapters, metadata. Uses WebVTT format.

üîå Embedded Content

10. <embed> - Embedded Content

Syntax:

<embed src="document.pdf" type="application/pdf" width="600" height="400">

Output:

Notes:

Self-closing tag. For external applications, plugins, or media. Use sparingly - prefer specific elements like <video>, <audio>.

11. <object> - Generic Object

Syntax:

<object data="movie.swf" type="application/x-shockwave-flash" width="400" height="300">
    <param name="movie" value="movie.swf">
    <p>Flash content not supported.</p>
</object>

Output:

Notes:

More flexible than <embed>. Can contain fallback content and parameters. Often used for Flash (now deprecated).

12. <param> - Object Parameter

Syntax:

<param name="autoplay" value="true">
<param name="loop" value="false">

Output:

See usage in <object> example above.

Notes:

Self-closing tag. Defines parameters for <object> elements. Name-value pairs for configuration.

13. <iframe> - Inline Frame

Syntax:

<iframe src="https://www.example.com" width="600" height="400" title="Example Website">
    <p>Your browser does not support iframes.</p>
</iframe>

Output:

Notes:

Embeds another HTML document. Security considerations with sandbox attribute. title required for accessibility.

üó∫Ô∏è Image Maps

14. <map> - Image Map

Syntax:

<img src="planets.jpg" alt="Planets" usemap="#planetmap">
<map name="planetmap">
    <area shape="circle" coords="90,58,3" href="mercury.html" alt="Mercury">
    <area shape="circle" coords="124,58,8" href="venus.html" alt="Venus">
</map>

Output:

Notes:

Creates clickable areas on images. name attribute matches usemap on img. Rarely used in modern web design.

15. <area> - Image Map Area

Syntax:

<area shape="rect" coords="0,0,100,100" href="link.html" alt="Rectangle area">
<area shape="circle" coords="50,50,25" href="link.html" alt="Circle area">
<area shape="poly" coords="0,0,50,25,0,50" href="link.html" alt="Polygon area">

Output:

See usage in <map> example above.

Notes:

Self-closing tag. Shapes: rect, circle, poly, default. Coordinates depend on shape type.

üé® Graphics Elements

16. <canvas> - Drawing Canvas

Syntax:

<canvas id="myCanvas" width="300" height="200">
    Your browser does not support canvas.
</canvas>

Output:

Your browser does not support canvas.

Notes:

HTML5 element for drawing graphics via JavaScript. Requires scripting to display content.

17. <svg> - Scalable Vector Graphics

Syntax:

<svg width="200" height="100">
    <circle cx="50" cy="50" r="40" fill="red"></circle>
    <rect x="100" y="10" width="80" height="80" fill="blue"></rect>
</svg>

Output:

Notes:

Vector graphics format. Scalable, searchable, and styleable with CSS. Can be inline or external file.

üí° Media Best Practices

  • Always provide alt text for images
  • Use multiple source formats for audio/video compatibility
  • Include fallback content for unsupported elements
  • Optimize media file sizes for web delivery
  • Use responsive images with srcset for different screen sizes
  • Be cautious with autoplay - many browsers block it
  • Provide captions and transcripts for accessibility
