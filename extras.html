‚ö†Ô∏è Deprecated Elements (Avoid in Modern HTML)

Note: These elements are deprecated and should not be used in modern web development. They're included here for educational purposes and understanding legacy code.

1. <marquee> - Scrolling Text (Deprecated)

Syntax:

<marquee direction="left" behavior="scroll" scrollamount="3">
    This text scrolls from right to left
</marquee>

Output:

This text scrolls from right to left

Notes:

Deprecated! Creates scrolling text. Poor for accessibility and UX. Use CSS animations instead.

2. <center> - Center Alignment (Deprecated)

Syntax:

<center>This text is centered</center>

Output:

This text is centered

Notes:

Deprecated! Use CSS text-align: center instead.

3. <font> - Font Styling (Deprecated)

Syntax:

<font face="Arial" size="4" color="red">Styled text</font>

Output:

Styled text

Notes:

Deprecated! Use CSS for font styling: font-family, font-size, color.

4. <basefont> - Base Font (Deprecated)

Syntax:

<basefont face="Times" size="3" color="black">

Output:

Not displayed - sets default font for document

Notes:

Deprecated! Self-closing tag. Use CSS body selector for default font styling.

5. <acronym> - Acronym (Deprecated)

Syntax:

<acronym title="World Wide Web">WWW</acronym>

Output:

WWW

Notes:

Deprecated! Use <abbr> instead for both abbreviations and acronyms.

6. <applet> - Java Applet (Deprecated)

Syntax:

<applet code="MyApplet.class" width="300" height="200">
    Java applet not supported
</applet>

Output:

Java applets are no longer supported by modern browsers

Notes:

Deprecated! Java applets removed from modern browsers. Use modern web technologies instead.

7. <blink> - Blinking Text (Deprecated)

Syntax:

<blink>This text would blink</blink>

Output:

This text would blink (No longer works in modern browsers)

Notes:

Deprecated! Never standardized, removed from browsers. Poor for accessibility.

8. <frame> and <frameset> - Frames (Deprecated)

Syntax:

<frameset cols="50%,50%">
    <frame src="left.html">
    <frame src="right.html">
</frameset>

Output:

Frames are not supported in HTML5

Notes:

Deprecated! Removed in HTML5. Use <iframe> for embedding or CSS/JavaScript for layouts.

10. <bdo> - Bidirectional Override

Syntax:

<bdo dir="rtl">This text will be right-to-left</bdo>
<bdo dir="ltr">This text will be left-to-right</bdo>

Output:

This text will be right-to-left

This text will be left-to-right

Notes:

Overrides text direction. dir attribute required: "ltr" (left-to-right) or "rtl" (right-to-left).

üîß Template and Web Components

11. <template> - Template Element

Syntax:

<template id="my-template">
    <div class="card">
        <h3>Template Title</h3>
        <p>Template content goes here</p>
    </div>
</template>

Output:

Template content is not rendered until activated by JavaScript

Notes:

HTML5 element. Content not rendered until cloned and inserted via JavaScript. Used in web components.

12. <slot> - Content Slot

Syntax:

<template id="user-card">
    <div class="user-card">
        <slot name="avatar">Default Avatar</slot>
        <slot name="name">Default Name</slot>
        <slot>Default content</slot>
    </div>
</template>

Output:

Slots are placeholders used in web components and shadow DOM

Notes:

HTML5 element. Placeholder for content in web components. name attribute creates named slots.

üîç Miscellaneous Elements

16. <noscript> - No Script Fallback

Syntax:

<noscript>
    <p>This website requires JavaScript to function properly.</p>
    <p>Please enable JavaScript in your browser.</p>
</noscript>

Output:

(Content above only shows if JavaScript is disabled)

Notes:

Content displayed only when JavaScript is disabled or unavailable. Important for accessibility.

17. <wbr> - Word Break Opportunity

Syntax:

supercalifragilisticexpiali<wbr>docious

Output:

supercalifragilisticexpialidocious

Notes:

Self-closing tag. Suggests where browser can break long words if needed for line wrapping.

18. <data> - Machine-Readable Data

Syntax:

<data value="12345">Product ABC</data>
<data value="2024-01-15">January 15, 2024</data>

Output:

Item: Product ABC

Date: January 15, 2024

Notes:

HTML5 element. Associates machine-readable value with human-readable content. Similar to <time> but for any data.

19. <keygen> - Key Generator (Deprecated)

Syntax:

<keygen name="security" challenge="randomstring" keytype="rsa">

Output:

Keygen element was removed from HTML5 specification

Notes:

Deprecated! Was used for generating public-private key pairs. Removed due to security concerns.

20. <menuitem> - Menu Item (Deprecated)

Syntax:

<menuitem label="Copy" onclick="copy()"></menuitem>

Output:

Menuitem element was removed from HTML5 specification

Notes:

Deprecated! Was intended for context menus. Never widely supported, removed from spec.

üé® Styling-Related Deprecated Elements

21. <s> vs <strike> - Strikethrough

Syntax:

<s>This text is struck through (HTML5)</s>
<strike>This text is struck through (deprecated)</strike>

Output:

This text is struck through (HTML5)

This text is struck through (deprecated)

Notes:

<s> is valid HTML5 for content no longer accurate. <strike> is deprecated - use <s> or CSS text-decoration.

22. <tt> - Teletype Text (Deprecated)

Syntax:

<tt>Monospace text</tt>

Output:

Monospace text

Notes:

Deprecated! Use <code>, <kbd>, <samp>, or CSS font-family: monospace instead.

23. <dir> - Directory List (Deprecated)

Syntax:

<dir>
    <li>file1.txt</li>
    <li>file2.txt</li>
</dir>

Output:

  • file1.txt
  • file2.txt
    •

    Notes:

    Deprecated! Use <ul> instead for directory listings.
