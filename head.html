1. <!DOCTYPE> Declaration

Syntax:

<!DOCTYPE html>

Output:

This declaration tells the browser this is an HTML5 document (already applied to this page)

Notes:

Must be the very first line in an HTML document. HTML5 simplified this from complex SGML declarations.

2. <html> Element

Syntax:

<html lang="en">
    <!-- All page content goes here -->
</html>

Output:

Root element containing all page content (wraps this entire page)

Notes:

The lang attribute helps screen readers and search engines understand the page language.

3. <head> Element

Syntax:

<head>
    <!-- Metadata goes here -->
</head>

Output:

Contains metadata not displayed on the page (like the head section of this page)

Notes:

Everything in the head is for browsers, search engines, and other tools - not visible to users.

4. <title> Element

Syntax:

<title>HTML Head Elements - Educational Website</title>

Output:

Check your browser tab - that's the title in action!

Notes:

Appears in browser tabs, bookmarks, and search engine results. Should be descriptive and unique.

5. <meta> Elements

Character Set

Syntax:

<meta charset="UTF-8">

Output:

Enables proper display of special characters: √©, √±, ‰∏≠Êñá, ÿßŸÑÿπÿ±ÿ®Ÿäÿ©, üåü

Notes:

UTF-8 supports all Unicode characters. Should be first meta tag.

Viewport

Syntax:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Output:

Makes this page responsive on mobile devices

Notes:

Essential for responsive design. Controls how page scales on different devices.

Description

Syntax:

<meta name="description" content="Learn HTML Head Elements">

Output:

Appears in search engine results (not visible on page)

Notes:

Should be 150-160 characters. Influences search engine rankings.

Keywords

Syntax:

<meta name="keywords" content="HTML, head, meta, tags">

Output:

Not visible to users (mostly ignored by modern search engines)

Notes:

Largely deprecated for SEO but still used by some internal search systems.

Author

Syntax:

<meta name="author" content="HTML Education Team">

Output:

Specifies the page author (not visible to users)

Notes:

Useful for content management and attribution.

Refresh

Syntax:

<meta http-equiv="refresh" content="30">

Output:

Would refresh this page every 30 seconds (not applied here for usability)

Notes:

Can redirect to another page: content="5;url=https://example.com"

6. <base> Element

Syntax:

<base href="https://example.com/" target="_blank">

Output:

Sets default URL and target for all relative links on the page

Notes:

Only one base element allowed per document. Affects all relative URLs.

7. <link> Element

Syntax:

<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">
<link rel="canonical" href="https://example.com/page">

Output:

Links external resources (CSS files, icons, etc.) - not visible but affects page appearance

Notes:

Self-closing tag. Common rel values: stylesheet, icon, canonical, preload, prefetch.

8. <style> Element

Syntax:

<style>
    body { font-family: Arial; }
    h1 { color: blue; }
</style>

Output:

Contains CSS styles for the page (this page uses no CSS to focus on HTML)

Notes:

Internal CSS. External stylesheets via <link> are generally preferred.

9. <script> Element

Syntax:

<script>
    console.log("Hello World!");
</script>
<script src="script.js"></script>

Output:

Executes JavaScript code (check browser console for "Hello World!" message)

Notes:

Can contain inline code or link external files. Placement affects when code runs.

10. <noscript> Element

Syntax:

<noscript>
    <p>JavaScript is disabled in your browser.</p>
</noscript>

Output:

(If you see a message above, JavaScript is disabled)

Notes:

Content only displays when JavaScript is disabled or unavailable.

üí° Head Elements Best Practices

  • Always include charset meta tag first
  • Use viewport meta for responsive design
  • Write descriptive, unique titles
  • Keep descriptions under 160 characters
  • Use semantic and descriptive meta information
